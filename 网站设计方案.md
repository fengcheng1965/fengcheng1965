# 苏顺植保网站设计方案

## 一、设计理念

基于图片分析报告，本网站设计将延续"苏顺植保"的核心视觉特征：
- **绿色为主色调**：体现农业、自然、环保理念
- **现代简洁风格**：符合当前网页设计趋势
- **专业可信形象**：传递植保行业的专业性
- **清新自然氛围**：营造舒适的浏览体验

## 二、色彩系统设计

### 主色调方案

```css
/* 主色调 - 绿色系 */
--primary-green: #2E7D32;      /* 深绿色 - Logo、导航栏 */
--secondary-green: #43A047;    /* 中绿色 - 按钮、强调元素 */
--accent-green: #66BB6A;       /* 翠绿色 - 悬停效果、图标 */
--light-green: #A5D6A7;        /* 浅绿色 - 背景、分隔线 */

/* 辅助色 - 蓝色系 */
--primary-blue: #1976D2;       /* 深蓝色 - 信任、科技 */
--light-blue: #BBDEFB;         /* 浅蓝色 - 背景点缀 */

/* 警示色 - 橙色系 */
--warning-orange: #FF9800;     /* 橙色 - 注意事项、提示 */

/* 中性色 */
--text-primary: #212121;       /* 深灰 - 主要文字 */
--text-secondary: #757575;     /* 中灰 - 次要文字 */
--background-light: #F5F5F5;   /* 浅灰 - 页面背景 */
--white: #FFFFFF;              /* 白色 - 卡片背景 */
--border-color: #E0E0E0;       /* 边框颜色 */
```

### 色彩应用规范

| 元素类型 | 推荐颜色 | 应用场景 |
|---------|---------|----------|
| Logo/品牌标识 | --primary-green | 网站Logo、标题 |
| 导航栏 | --primary-green | 顶部导航背景 |
| 按钮（主要） | --secondary-green | 提交按钮、CTA按钮 |
| 按钮（次要） | --light-green | 辅助按钮 |
| 链接文字 | --primary-blue | 超链接文字 |
| 主要文字 | --text-primary | 标题、正文 |
| 次要文字 | --text-secondary | 说明文字、备注 |
| 背景色 | --background-light | 页面整体背景 |
| 卡片背景 | --white | 内容区块背景 |
| 边框/分隔线 | --border-color | 元素边框 |

## 四、响应式设计方案

### 断点设置

```css
/* 超小屏幕（手机，小于576px） */
@media (max-width: 575.98px) { }

/* 小屏幕（平板，576px起） */
@media (min-width: 576px) and (max-width: 767.98px) { }

/* 中等屏幕（桌面，768px起） */
@media (min-width: 768px) and (max-width: 991.98px) { }

/* 大屏幕（桌面，992px起） */
@media (min-width: 992px) and (max-width: 1199.98px) { }

/* 超大屏幕（桌面，1200px起） */
@media (min-width: 1200px) { }
```

### 移动端适配策略

1. **导航栏**：转为汉堡菜单
2. **网格布局**：从多列变为单列
3. **字体大小**：适当缩小以适应小屏幕
4. **图片**：使用响应式图片，加载适合屏幕尺寸的图片
5. **按钮**：增大点击区域，最小44x44px
6. **间距**：适当减少留白以节省空间

## 六、图片使用规范

### 1. 图片格式
- **优先使用**：WebP（现代浏览器支持，文件小）
- **备用格式**：JPG（照片类）、PNG（透明背景）

### 2. 图片优化
- 压缩图片文件大小
- 使用响应式图片（srcset）
- 添加懒加载

### 3. 图片内容
- 产品图片：清晰展示产品外观、细节
- 场景图片：展示产品使用场景、效果对比
- 团队图片：专业形象照
- 案例图片：实际应用案例展示
