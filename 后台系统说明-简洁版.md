## 📖 查看客户信息流程（待开发）

### 操作步骤

**1. 登录系统**
```
访问：http://localhost:3000/admin/login
输入：用户名和密码
点击：登录按钮
```

**2. 进入客户管理**
```
顶部导航栏 → 客户管理 → 客户列表
访问路径：http://localhost:3000/admin/customers
```

**3. 筛选和搜索**
```
搜索框：输入客户名称、电话或邮箱
筛选条件：选择客户状态、日期范围
点击：搜索按钮
```

**4. 查看客户详情**
```
在列表中找到目标客户
点击：查看按钮
查看：基本信息、订单记录、互动记录
```
