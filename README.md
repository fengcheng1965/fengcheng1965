# 苏顺植保网站

一个专业的农业植保服务网站，提供产品展示、在线咨询和业务沟通功能。

## 项目简介

苏顺植保网站是一个集产品展示、在线咨询、业务沟通于一体的综合性农业服务平台。网站采用现代化的响应式设计，支持多种设备访问，为用户提供便捷的产品查询和咨询服务。

## 快速开始

### 本地运行

1. 克隆项目到本地：
```bash
git clone https://github.com/fengcheng1965/fengcheng1965.git
cd fengcheng1965
```

2. 使用本地服务器运行：

**方法1：使用Python**
```bash
# Python 3
python -m http.server 8000

# Python 2
python -m SimpleHTTPServer 8000
```

**方法2：使用Node.js**
```bash
# 安装http-server（首次使用）
npm install -g http-server

# 运行服务器
http-server -p 8000
```

**方法3：使用PHP**
```bash
php -S localhost:8000
```

3. 在浏览器中访问：
```
http://localhost:8000
```
