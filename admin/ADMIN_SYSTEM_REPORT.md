## 🎯 项目目标

为后端系统开发一套直观、高效的可视化操作界面，实现对后端服务、数据模型、接口配置、权限管理等核心功能的图形化管理。该界面支持通过拖拽、点击等交互方式完成后端配置，提供实时预览和验证功能，并确保操作流程符合软件工程最佳实践。系统包含数据可视化模块，能够直观展示后端性能指标、资源使用情况及接口调用统计数据。

2. 访问 http://localhost:8081

5. 查看实时生成的代码

1. 导航到"实时监控"页面
2. 点击"开始监控"按钮
